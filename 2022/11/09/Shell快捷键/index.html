Shell快捷键

Shell快捷键：

Ctrl + a ————切换到命令行开始

Ctrl + e ————切换到命令行末尾

Ctrl + l ————清除屏幕内容，等同于clear命令

Ctrl + u ————清除/剪切光标之前的内容

Ctrl + k ————清除/剪切光标之后的内容

Ctrl + y ————粘贴刚才所删除的字符

Ctrl + r ————在历史命令中查找，输入关键字就能调出以前的命令

Ctrl + c ————终止命令

Ctrl + d ————退出shell，相当于logout命令

行首 行尾 前一个单词 后一个单词
移动 ctrl + a (home) ctrl + e (end) ctrl + 方向键左 （alt + f） ctrl + 方向键右 (alt + b)
删除 ctrl + u (删当前到行首) ctrl + k (删当前到行尾) ctrl + w (删除至词首) alt + d (删除至词尾)
